## 🌐 API Documentation
Base URL: https://driver-ahv6.onrender.com/api

```
https://driver-ahv6.onrender.com/api
```

### Authentication
JWT token stored locally, expires after 7 days.

```
// Add to headers for protected endpoints headers: {'Authorization': 'Bearer $token'}
```

### 1. Authentication Endpoints
#### POST /api/auth/register

```
/api/auth/register
```

Register new driver account.

```
name
```


```

```


```

```

```


```

Request:

```
{ "name": "Nguyễn Văn A", "phone": "0901234567", "password": "123456", "carType": "Toyota Vios", "carYear": "2020" }
```

Response (201):

```
{ "message": "Đăng ký thành công. Vui lòng chờ admin phê duyệt.", "user": { "id": "60d21b4667d0d8992e610c85", "name": "Nguyễn Văn A", "phone": "0901234567", "status": "pending" } }
```

### 3. Requests Endpoints
#### POST /api/requests

```
/api/requests
```

Create ride request (public, no auth).
Request:

```
{ "name": "Lê Văn C", "phone": "0923456789", "startPoint": "Hà Nội", "endPoint": "Hải Phòng", "price": 500000, "note": "Đi lúc 8h sáng", "region": "north" }
```

Response (201):

```
{ "message": "Request created successfully", "request": { "_id": "60d21b4667d0d8992e610c88", "status": "waiting", ... } }
```


#### GET /api/requests

```
/api/requests
```

Get all requests.
Query Params:

```
status
```


```

```


```

```


```

```



```
/api/requests/my-requests
```

Get current user's requests. Auth required.

### Key Implementation Notes
1. Phone Masking:

```
String maskPhone(String phone, {bool isLoggedIn = false}) { if (isLoggedIn) return phone; if (phone.length >= 10) { return '${phone.substring(0, 3)} xxxx ${phone.substring(phone.length - 3)}'; } return phone; }
```

1. Region Enum:

```
enum Region { north, central, south } extension RegionExtension on Region { String get label { switch (this) { case Region.north: return 'Miền Bắc'; case Region.central: return 'Miền Trung'; case Region.south: return 'Miền Nam'; } } }
```

1. 63 Vietnamese Provinces:

```
final Map<Region, List<String>> provincesByRegion = { Region.north: ['Hà Nội', 'Hải Phòng', 'Hải Dương', ...], Region.central: ['Đà Nẵng', 'Huế', 'Quảng Nam', ...], Region.south: ['TP. Hồ Chí Minh', 'Bình Dương', 'Đồng Nai', ...], };
```

1. API Service:

```
class ApiService { static const baseUrl = 'https://driver-ahv6.onrender.com/api'; Future<List<Driver>> getDrivers({Region? region}) async { final url = region != null ? '$baseUrl/drivers?region=${region.name}' : '$baseUrl/drivers'; final response = await http.get(Uri.parse(url)); // Parse response... } }
```

1. Secure Token Storage:

```
final storage = FlutterSecureStorage(); await storage.write(key: 'token', value: token); await storage.write(key: 'user', value: jsonEncode(user));
```
